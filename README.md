
# Sistema de Gestión de Concesionario de Autos

## Descripción
El **Sistema de Gestión de Concesionario de Autos** es una solución integral diseñada para mejorar la eficiencia y precisión en la gestión del inventario de vehículos, optimizar el procesamiento de ventas, gestionar clientes y empleados, y ofrecer un servicio al cliente mejorado mediante el uso de inteligencia artificial (IA). Este sistema está pensado para concesionarios que buscan modernizar sus operaciones y brindar una experiencia superior a sus clientes.

## Alcance del Proyecto
El sistema cubre las siguientes áreas clave:
1. **Gestión de Automóviles**: Control completo del inventario de vehículos, incluyendo agregar, editar, eliminar y buscar automóviles por características como marca, modelo, año, precio, etc.
2. **Procesamiento de Ventas**: Registro de ventas, que se guardan automáticamente en el historial del cliente.
3. **Gestión de Clientes**: Registro de clientes y seguimiento de su historial de compras.
4. **Gestión de Empleados**: Administración de empleados, con la capacidad de agregar nuevos empleados.
5. **Servicio al Cliente con IA**: Implementación de un asistente virtual basado en IA para responder preguntas frecuentes y brindar recomendaciones.

## Características Principales
- **Inventario de Vehículos**:
  - Agregar, editar y eliminar automóviles.
  - Búsqueda avanzada por marca, modelo, año, precio, etc.
- **Ventas**:
  - Registro de ventas con detalles del cliente, vehículo y empleado.
  - Las ventas se guardan automáticamente en el historial del cliente.
- **Clientes**:
  - Registro y actualización de información de clientes.
  - Historial de compras y preferencias.
- **Empleados**:
  - Gestión de empleados (agregar nuevos empleados).
  - Todos los empleados tienen las mismas funciones y permisos.
- **Servicio al Cliente con IA**:
  - Asistente virtual para responder preguntas frecuentes.
  - Recomendaciones personalizadas de vehículos.

## Tecnologías Utilizadas
- **Lenguaje de Programación**: Java
- **Framework**: NetBeans
- **Base de Datos**: Pendiente (por definir)
- **Frontend**: Java Swing
- **Control de Versiones**: Git/GitHub

## Requisitos del Sistema
- Java JDK 11 o superior
- NetBeans IDE (recomendado)
- Git para control de versiones
- Base de datos (pendiente de definición)
